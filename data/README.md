# Dataset Information

The original raw and processed datasets used in this project are not included in this repository due to GitHub file size limitations.

The project was developed using logistics shipment data containing:

* Source and Destination Hubs
* Route Information
* Travel Distance and Duration
* ETA Records
* Shipment Status Information
* Network Analytics Features

The processed datasets were used for:

* Feature Engineering
* ETA Prediction Modeling
* Network Analytics
* Bottleneck Detection
* Dashboard Generation

All project notebooks, trained models, reports, and dashboard code are included in this repository.
